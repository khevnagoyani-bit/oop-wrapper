# oop-wrapper

👔 Employee Management System

«A menu-driven Python application that demonstrates the power of Object-Oriented Programming (OOP) through employee record management.»

Employee Management System is a command-line application built with Python to showcase the core principles of Object-Oriented Programming. The project allows users to create and manage different types of people—Person, Employee, and Manager—while demonstrating inheritance, method overriding, and polymorphism in a practical way.

---

🌟 Overview

Object-Oriented Programming is one of the most important concepts in software development. This project provides a hands-on implementation of OOP by organizing employee information into a class hierarchy.

The application enables users to create different objects, store their information, and display their details using overridden methods. It is an excellent learning project for understanding how inheritance and polymorphism simplify code organization and improve reusability.

---

✨ Key Features

- 👤 Create a Person object
- 💼 Create an Employee object
- 👨‍💼 Create a Manager object
- 📋 Display details of each object
- 🧬 Demonstrates multi-level inheritance

🚀 Getting Started

Requirements

- Python 3.x

Run the Application

1. Save the source code as "employee_management.py".
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

python employee_management.py
